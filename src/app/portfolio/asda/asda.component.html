Project Description

Walmart is a retailing website that sells almost all the products both online and in stores. And in the current project the aim was to add new business requirements, update & enhance the existing functionalities like simplify adding to cart, increase or decrease the quantity, quick look at other relavent products, new click and collect feature, showing dynamic ads like new products and related products to consider, etc. Without effecting other modules and optimise the already existing website there by gradually revamping the entire child website but retaining the branding of the parent with enhanced user experience with strict web accessibility and as per W3C standards.

Responsibilities
  • Involving in multi-disciplinary meetings and gathering the project and UI requirements.
  • Discussion/updating & support teams across locations US, UK and India
  • Creating new workflows.
  • Designing new UIs(if any) based on wireframes provided by UX Team.
  • Develop HTML UIs as per the designs.
  • Bug fixing by resolving all the cross browser issues from IE6 to IE9, FF3.6+, Safari4+.
  • Implementing web accessibility in all pages.
  • Website testing, user testing the new modules.
  • Website performance and optimization.
‍Challenges

Similar to every other IT project we have had a few challanges which as a team we could easily find solutions. Specifically to mention we faced a ‍Challenge where we were unable to decide on going with a specific feature for a set postcode then we had A/B Testing implmented and through anlytics got our issue solved.
